Instagram Story Creator - 利用ガイド
Instagramのストーリーズ用に、店舗の予約状況画像を簡単に作成できるツールです。

使い方
アプリを開く

index.html
 をブラウザ（Chrome, Edgeなど）で開いてください。
テーマの選択

画面左上のプルダウンから、お好みのテーマを選択できます。
健康 (Health): 緑 (デフォルト)
活力 (Vitality): オレンジ
安らぎ (Peace): 青
リラックス (Relax): 紫
日付の設定

デフォルトで「今日」の日付が選択されています。
平日: 午後は 15:00 ～ 19:30 まで表示されます。
土日: 午後は 15:00 ～ 17:30 まで（18:00以降は自動的に非表示）になります。
予約状況の入力

各時間のプルダウンから状況を選択します：
〇: 余裕あり
×: 空きなし
△: 要相談
店舗IDの入力

「店舗ID (@StoreName)」欄にご自身のInstagram IDや店舗名を入力してください。
入力内容は自動的に保存され、次回以降は入力の手間が省けます。
コメントの入力

自由記述欄にメッセージを入力できます。
画像の保存

「画像を保存」ボタンをクリックすると、作成された画像がダウンロードされます。
カスタマイズについて
時間枠の設定などは 
script.js
 内で管理されています。
